## Features
- React / Redux / React-Router 
- 热加载 / browser-sync / 开发环境支持 redux devtools
- minify / chunkhash / trackJS 生产环境构建
- es6/webapck
- sass
- Ant Design
- isomorphic-fetch
- mock data
- ...

## Usage
### 安装
```
git clone git@github.com:ryan-hung/react-backend.git demo
cd demo
npm install

# 启动开发环境
npm run start
# 启动 mock 服务
npm run start:mock
```

### npm 脚本

```shell
# 开发环境启动
npm run start 

# 生产环境启动
npm run start:prod

# 生产环境构建
npm run build

# mock 服务
npm run start:mock

# 测试
npm run test

```
